Love of Tales

  • About

    Love of Tales is an online resource for The Lord of the Rings LCG by Fantasy Flight Games. The source code is available on Github.

  • Builder

    The deck builder was built by Dirk Meijer, and is maintained over at The Rivendell Councilroom. The deck builder represents a large portion of the code for Love of Tales, and this website would likely not exist without Dirk's deckbuilder. You can buy him a drink here:

    To build a deck, simply search for the card you want to add using the options on the left of the builder page, and click the button indicating the number of copies you would like to add. You can name and save the deck using the textbox and button in the top-right, and it will then be saved to the cloud and accessible from any device. Suggestions will appear as you are building your deck. These can be hidden using the button at the top of the suggestions column. The suggestion algorithm is continuously in development, so some strange suggestions may appear. Individual suggestions can be blocked by clicking the X next to the suggested card. The list of blocked suggestions is reset when the page is refreshed or when the 'Clear Blocked Suggestions' button is clicked.

  • Sets

    In the Sets tab, you can select which sets you own. Only cards from these sets will show up in the Builder tab. The packs you've selected are stored in your browser data and will be remembered for subsequent visits to the deckbuilder, but may need to be re-selected when moving to a different device.

  • My Decks

    In My Decks tab, you can view the decks you've made (or are still working on) and load decks you've made in the past. Beware that loading or uploading decks will overwrite the current deck in the builder without prompt, so save your deck before loading another deck (or open up a new tab in your browser), or it will be lost.
    Saved decks can be downloaded, either in a text file that contains markup for both Reddit (MarkDown) and BoardGameGeek (BBCode) or as an OCTGN deck. Decks can also be uploaded. Note that uploading Does not automatically save the deck, it only loads it into the Builder.
    These decks are stored on the cloud and are accessible by any device connected to the internet. To interact with a deck, click on it to highlight it, then click on one of the buttons. You can export the deck to text format or OCTGN format, load the deck into the builder, or open up the info page for the deck. Decks cannot currently be deleted from Love of Tales. It could easily be implemented, but I have currently decided against it for a couple reasons. First, each deck you make has a unique url that can be shared. By deleting the deck, it would cause a broken url for anyone that may click on your link in the future. Second, the decks in Love of Tales are intimately connected to the Quest Log. When you log a quest, it will link to all the decks used to beat that quest. Deleting a deck that was used in a quest would then break the links in not just your quest log, but anyone else's quest log that used your deck to complete a quest. So if you want to modify a deck, I recommend just appending a suffix like 'v2.0' and saving it as a new deck.

  • My Logs

    In the My Logs tab, you log a quest. You must input a Date, the quest you completed, whether it was a Success or Failure, the score, and at least one deck ID. Deck IDs can be found in the My Decks tab or in the url for the Deck Info page for a deck. Once you save a log it will appear in the 'My Logs' section with links to the decks used to beat the quest.

  • Community

    In the Community tab, you can perform a search on all the decks in the database. You can search using a portion of the deck title, or search for decks that have beaten a specific quest. You can search for decks that contain the card that is currently being displayed in the card preview in the sidebar. You can also limit the results to only show decks that can be built using the sets you've selected over in the 'Sets' tab.
